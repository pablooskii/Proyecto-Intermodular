Introducción

Este proyecto tiene como objetivo presentar la estructura y datos de la base de datos PRÁCTICA INTERMODULAR, que contiene información sobre categorías de plataformas digitales, las plataformas mismas y los planes de suscripción que ofrecen.

Se muestran las tablas HTML con los datos correspondientes y se implementa una interacción con JavaScript para mejorar la experiencia del usuario.
